## 1. Page Purpose & Strategic Role

The cluster page is the workhorse content template — 24+ pages across the site use this layout. It serves a single focused topic with deep educational content, then routes visitors toward comparison pages (for the undecided) or package pages (for the ready-to-buy). Anonymous LLC is the flagship cluster under the Privacy pillar and the single highest-value SEO target on the site (5,500–9,500 monthly searches for anonymous LLC terms).

Cluster pages sit in the middle of the funnel: Pillar → **here** → Comparison or Package. They must educate thoroughly while always keeping the exit-to-conversion visible.

### How Cluster Pages Differ From Pillar Pages

| Element | Pillar Page | Cluster Page |
|---------|-------------|--------------|
| Scope | Broad overview of an entire service category | Deep dive on one specific topic |
| Content depth | Scanner overviews with expandable sections | Full long-form content (scanners still get structure) |
| Navigation | Cluster grid (routes OUT to clusters) | Related clusters sidebar (links to SIBLINGS) |
| CTAs | Soft CTAs + package shortcuts | Progression CTAs ("Compare States →") + package CTAs |
| Sidebar | Package shortcut + cluster list | Related clusters + parent pillar link + package shortcut |

### Target Audiences

| Audience | Entry Point | Expected Path |
|----------|-------------|---------------|
| Organic searcher | Google: "anonymous LLC", "how to form anonymous LLC" | This page → FAQ or compare → Gold package |
| Pillar visitor | Privacy pillar cluster grid → clicked "Anonymous LLC" | This page → compare states → Gold package |
| Cross-pillar referral | AP or Formation page → "add privacy" cross-sell | This page → nominee services cluster → Gold package |
| Informed researcher | Direct URL or bookmark | Scans content, jumps to FAQ or packages |

## 5. Long-Form Content Sections

The main body of the page. This is long-form educational content structured with clear H2/H3 hierarchy for scanners, but with enough depth for researchers. Each section uses the 80/20 model: concise visible content, with optional expandable detail.

### Section 1: "What is an Anonymous LLC?"

| Element | Content |
|---------|---------|
| H2 | "What is an Anonymous LLC?" |
| Content | 3–4 paragraphs explaining: an anonymous LLC is a limited liability company formed in a state that does not require public disclosure of members or managers. The LLC is real, legal, and fully functional — the only difference is that the owner's name is absent from the state's public database. Wyoming and Nevada are the two primary states that allow this structure. |
| Key Point Callout | Highlighted box: "An anonymous LLC is not a special entity type — it's a standard LLC formed in a privacy-friendly state with nominee services replacing your name on public filings." |
| Expandable Detail | "How does public disclosure work? →" — expands to 200–300 words covering which state records are public, what databases exist, how anyone can search them, and what information gets exposed without privacy measures. |

### Section 2: "How Anonymous LLCs Work"

| Element | Content |
|---------|---------|
| H2 | "How Anonymous LLCs Work" |
| Content | Step-by-step explanation (numbered list or visual flow): |

**Steps:**

| Step | Title | Description |
|------|-------|-------------|
| 1 | "Choose a Privacy-Friendly State" | "Wyoming or Nevada. Both allow LLC formation without public member/manager disclosure. Wyoming is generally preferred for lower costs and stronger statutory protections." |
| 2 | "Appoint Nominee Officers" | "A nominee officer or director appears on the Articles of Organization filed with the state. This person's name — not yours — appears in the public record." |
| 3 | "File Formation Documents" | "The Articles of Organization are filed with the Secretary of State. Your registered agent (Incorporate123) handles this. The nominee's name is on the filing." |
| 4 | "Operate With Privacy Intact" | "All annual reports, amendments, and public filings use the nominee. Your connection to the entity is documented only in private operating agreements — not in any state database." |

| Expandable Detail | "What about the IRS? →" — expands to explain that the IRS knows the beneficial owner via EIN application and tax returns, but IRS records are not publicly searchable. BOI reporting goes to FinCEN, also not public. |
