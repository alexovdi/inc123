## Renewal Pricing Model — CRITICAL CHANGE

This is the single most important pricing decision. The old site and prototype use fundamentally different renewal models.

| Aspect                | Old Site Model                                                            | Prototype Model                               | Impact                                        |
| --------------------- | ------------------------------------------------------------------------- | --------------------------------------------- | --------------------------------------------- |
| **Pricing structure** | Base service fee **PLUS** state fees (billed separately)                  | All-inclusive flat rate (state fees included) | Changes how customers perceive renewal cost   |
| **NV Gold Renewal**   | $1,375 + NV state fees (~$350 LLC / ~$650 Corp) = **$1,725–$2,025 total** | $525 (LLC) / $625 (Corp)                      | **MASSIVE reduction** if prototype is correct |
| **NV Silver Renewal** | $850 + NV state fees (~$350 LLC / ~$650 Corp) = **$1,200–$1,500 total**   | $325 (LLC) / $425 (Corp)                      | **MASSIVE reduction**                         |
| **NV Bronze Renewal** | $575 + NV state fees (~$350 LLC / ~$650 Corp) = **$925–$1,225 total**     | $225 (LLC) / $325 (Corp)                      | **MASSIVE reduction**                         |
| **WY Gold Renewal**   | $1,225 (includes WY state fees of $50)                                    | $525 (LLC) / $625 (Corp)                      | **~$600–$700 reduction**                      |
| **WY Silver Renewal** | $850 (includes WY state fees of $50)                                      | $325 (LLC) / $425 (Corp)                      | **~$425–$525 reduction**                      |

### The Question for David:

The prototype renewal prices ($225–$625) are dramatically lower than the old site prices ($575–$2,025). This could mean:

1. **The old prices were too high** and David wants to reduce renewals to retain clients and match the competitive analysis (WY Corporate Services charges ~$545+/yr for comparable service)
2. **The prototype prices are aspirational** and need adjustment upward
3. **The prototype prices exclude state fees** and need a "+state fees" note added

**We need David to confirm the actual renewal pricing structure before any documents go final.**

### Question: Should we keep age-tiered pricing or go flat?

**Recommendation:** Keep age-tiered pricing. A 2013 entity (13 years old) is genuinely more valuable than a 2020 entity (6 years old). Flat pricing either overcharges for newer entities or undercharges for older ones. The old model was:

| Age                | Price Range   |
| ------------------ | ------------- |
| 5–6 years (2020)   | $2,000–$2,500 |
| 10–12 years (2014) | $4,000        |
| 13+ years (2013)   | $4,500+       |
