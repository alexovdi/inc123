### [UPDATED] David's State Fee Data (from meeting)

David provided actual state fee numbers that affect renewal cost calculations:

| State   | Annual State Fees (approx.) | Notes                                                |
| ------- | --------------------------- | ---------------------------------------------------- |
| Wyoming | ~$100/yr                    | Relatively low; renewal pricing has more margin room |
| Nevada  | $425–$700/yr                | Significantly higher; must be factored into renewals |

**Impact:** NV renewals should logically be higher than WY renewals because Nevada's state fees alone are $425–$700/yr. The current prototype shows identical renewal prices across states, which is likely incorrect.

### The Question for David (Still Open):

The prototype renewal prices ($225–$625) are dramatically lower than the old site prices ($575–$2,025). With the state fee data above, we need David to confirm:

1. **Do prototype prices include state fees?** If so, NV Bronze renewal at $225 doesn't even cover NV state fees ($425–$700).
2. **Are these Inc123 service fees only?** If so, we need to add "+state fees" notes and adjust customer-facing totals.
3. **What are the actual renewal prices?** We need final numbers per package per state.

### Question: Should we keep age-tiered pricing or go flat?

**Recommendation:** Keep age-tiered pricing. A 2013 entity (13 years old) is genuinely more valuable than a 2020 entity (6 years old). Flat pricing either overcharges for newer entities or undercharges for older ones. The old model was:
