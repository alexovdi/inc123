### Package Structure Simplification

- **Simplified from ~15 packages to 3 core tiers** for both Nevada and Wyoming: Bronze, Silver, and Gold
- Eliminated complex variations (Bronze Plus, Silver Max, Gold Executive, Gold Max, etc.) by converting them to add-ons/upgrades
- California and Florida packages remain as specialized options

### Add-Ons & Upgrades Structure

- **Decided to use "upgrade" terminology** for consistency instead of "add-ons"
- **Service upgrades include:**
  - Privacy Services Upgrade (year-round nominee directors/officers)
  - Part-Time Private Suite Upgrade
  - Executive Virtual Office (VO) Upgrade
- **Government Filing Services add-ons** (separate category):
  - Foreign State Registration (available for all 50 US states)
  - Domestication Filing (Wyoming or Nevada)
  - Corporate DBA Filing

### Package Details & Corrections

- **Wyoming Silver and Gold both include basic virtual office**
- **All packages include:**
  - Initial state filings and fees
  - Corporate records and minutes
  - Registered agent services
  - Offshore records storage
- **Bank account offering clarified** as "Bank Account Opening Document Package" (not actual account setup)
- **EIN obtainment and custom operating agreement** included in Silver and Gold only (not Bronze)
- **Foreign state registration** requires both registration filing AND registered agent service in that state

### Website Design Review

- David approved the new website design direction
- Design described as bringing the site "into 2025, 2026 from 1985"
- **Homepage features reviewed:**
  - Business privacy pillars section
  - Asset protection and compliance messaging
  - Customer guidance wizard
- **Wizard flow demonstrated:**
  - Asks what brings user (business formation, asset protection, compliance)
  - Asks business type (online vs brick and mortar)
  - Asks about privacy needs
  - Asks which state they operate in
  - Recommends appropriate package based on answers
- **For foreign state operations:** wizard will offer appropriate add-on for state registration after package selection
- **Comparison table** will display the 8 main packages with key features from the spreadsheet

### Compliance & Content Organization

- **Registered agent moved under compliance section** in package details
- Corporate seal line item removed (line 26)
- Certificate of good standing removed (line 21)
- Compliance section includes: initial state filings and fees, corporate records and minutes, registered agent
- Nominee directors/officers language updated to include "managers" for LLC structures
