## Architecture Overview

This document presents the finalized ~75-page site architecture for the Incorporate123 website redesign. The architecture has been validated against: 392 crawled URLs mapped to destinations, 5 competitors analyzed for structural gaps, 3 user journeys confirmed through flow analysis, and content migration data identifying priorities.

### Architecture Summary

| Component | Pages | Purpose |
|---|---|---|
| Pillar Pages | 4 | Primary navigation hubs for each service category |
| Cluster Pages | 27 | Deep-dive pages supporting each pillar's topics |
| Comparison Pages | 7 | Decision-support tools for the consideration stage |
| Package Pages | 8 | Conversion endpoints with pricing and order CTAs |
| State Hub Pages | 4 | Cross-pillar geographic landing pages |
| Utility/Support | 9+ | Homepage, About, Contact, FAQ, Blog, Legal |
| Offshore Hub | 6 | International formation services |
| **TOTAL** | **~75** | Consolidated from 325 unique current URLs |

> **v3 changes:** Cluster count changed from 25 to 27 (added Entity Tax Guide under Formation, Tax Filing & Obligations under Compliance). Comparison count changed from 6 to 7 (added LLC vs S-Corp vs C-Corp Tax under Formation). Net total ~75 (up from ~71).

## Internal Linking Strategy

The pillar/cluster hierarchy is established through internal linking, not URL structure. All URLs are flat (no directory nesting).

| From | Links To | Link Placement |
|---|---|---|
| Pillar Pages | All related clusters + comparison pages + relevant packages | In-content links, sidebar nav, bottom CTAs |
| Cluster Pages | Parent pillar + sibling clusters + relevant comparison + package | Breadcrumb, sidebar, in-content, bottom CTA |
| Comparison Pages | Parent pillar + compared packages | In-content links, CTA buttons per option |
| Package Pages | Related packages (Also Consider) + parent pillar | Sidebar, comparison strip, trust section |
| State Hubs | All content for that state across all pillars | Organized by pillar with links to each |
