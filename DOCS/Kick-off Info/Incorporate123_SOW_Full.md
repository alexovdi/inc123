# STATEMENT OF WORK
## Incorporate123.co — Website Redesign & Redevelopment

**Prepared by:** BIT Studios
**Prepared for:** Incorporate123
**Date:** February 2026

---

## 1. Project Overview

This Statement of Work defines the scope, deliverables, timeline, and terms for the complete redesign and redevelopment of the Incorporate123.co website. The project transforms the current geography-first website into a modern, intent-first platform organized around four content pillars: Business Privacy, Asset Protection, Company Formation, and Compliance.

The approved strategy document serves as the foundational reference for all work described in this SOW.

**Target Audience:** Small business owners and entrepreneurs with $500k+ in assets who prioritize privacy and asset protection over lowest price.

### 1.1 Parties

**Client:** Incorporate123 ("Client")

**Provider:** BIT Studios ("Provider")

### 1.2 Project Objectives

- Reposition Incorporate123 from "Nevada incorporation company" to "Privacy, Asset Protection & Business Formation Specialists"
- Implement a pillar/cluster site architecture organized around customer intent, not geography
- Design and develop a modern, trust-building website that reflects 25 years of expertise
- Create a clear conversion flow: Education → Specifics → Comparison → Package → Conversion
- Build SEO authority in privacy and asset protection keywords where competitors have limited presence
- Restructure packages into base packages with optional add-ons to reduce decision fatigue

---

## 2. Scope of Work

The project encompasses a complete website redesign and redevelopment across seven phases. All work is based on the approved strategic framework including the four-pillar architecture, ~71-page site map, and intent-first positioning.

### 2.1 Current Site

The current site contains approximately 171 URLs across 15 template types. After removing duplicate content paths (~42 duplicate FAQ URLs), there are ~129 unique content pages. Key issues include geography-first navigation that buries differentiators, significant content duplication in the Learning Center (~111 FAQ items across overlapping category paths), template sprawl (15 types, many serving only 1–3 pages), and an outdated design that undermines trust. A detailed site inventory is maintained separately.

### 2.2 New Site Architecture

The redesigned site is organized around four content pillars based on customer intent, with approximately 71 pages across 7–8 custom WordPress templates.

**Content Pillars:**

| Pillar | URL | Purpose | Est. Pages |
|--------|-----|---------|------------|
| Business Privacy (Lead) | /privacy/ | Anonymous LLC, nominee services, CA/FL private incorporation | 8–10 |
| Asset Protection | /asset-protection/ | NV/WY asset protection, holding structures, offshore AP | 5–7 |
| Company Formation | /formation/ | Entity selection, state guidance, shelf companies | 7–9 |
| Compliance | /compliance/ | Registered agent, annual reports, foreign registration | 4–5 |

The remaining pages include the homepage, 8–10 package pages, 3–5 comparison pages, 2–4 state hub pages, 5–6 offshore/international pages (retained but de-emphasized), and 5–7 utility pages. Packages are restructured into base tiers with reusable add-ons (foreign registration, domestication, virtual office, additional nominee services).

**Custom WordPress Templates (7–8):**

| Template | Used By | Key Components |
|----------|---------|----------------|
| Homepage | 1 page | Hero + CTA, service blocks, testimonials, trust signals, pillar navigation |
| Pillar Page | 4 pages | Educational long-form, progressive disclosure, cluster links, CTAs |
| Cluster Page | 15+ pages | Focused topic content, FAQ accordions, related clusters, package CTAs |
| Package Detail | 8–10 pages | Pricing, feature list, order CTA, add-on upsells |
| Comparison Page | 3–5 pages | Side-by-side tables, decision-focused content, package links |
| State Hub | 2–4 pages | Cross-pillar content aggregation for a specific state |
| Resource/FAQ | 5–10 pages | Organized FAQ accordions replacing current Learning Center |
| Utility Page | 5–7 pages | Standard content, contact form variant |

All ~170 current URLs will be mapped to the new structure via 301 redirects, finalized during Phase 1 Discovery.

### 2.3 E-Commerce & Checkout

The current site includes cart-based ordering with package selection and add-ons. The redesigned site replaces this with a modern e-commerce experience built on WooCommerce (or equivalent), including: 8 base packages configured as products, add-on products, shopping cart and checkout flow, payment gateway integration (Stripe and/or PayPal), order confirmation emails, order management in WordPress admin, and basic client account functionality.

### 2.4 In Scope

- Complete site architecture redesign using pillar/cluster model
- UX design including wireframes for all page types (homepage, pillar, cluster, comparison, package, cart, checkout)
- Visual design system, high-fidelity mockups, and component library
- Content strategy, page outlines, and content briefs for all ~71 pages
- Homepage and pillar page copy, CTA copy, meta descriptions, image alt text, and checkout/email copy
- SEO strategy implementation including keyword mapping, schema markup, internal linking, and meta optimization
- Full-stack WordPress development with custom theme and WooCommerce integration
- E-commerce checkout with payment gateway (Stripe/PayPal)
- Responsive design for desktop and mobile
- Google Analytics 4 and Google Search Console configuration verification and optimization
- QA testing, cross-browser compatibility, and performance optimization
- Staging site shared for client review prior to launch
- 301 redirect mapping from ~170 current URLs to new structure
- Launch execution with 1–2 weeks post-launch monitoring (analytics verification, bug fixes, performance checks)

### 2.5 Out of Scope

The following items are not included in this SOW but may be added as separate engagements:

- Ongoing SEO monitoring and optimization (post-launch)
- Content marketing, blog posts, and ongoing content updates
- Paid advertising management (Google Ads, retargeting)
- Social media marketing and email campaigns
- Interactive calculators, configurators, or client portal
- Multi-language support
- Third-party integrations (CRM, accounting software)
- Hosting and ongoing maintenance (to be quoted separately)
