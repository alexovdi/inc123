# Incorporate123 — Domain Portfolio Discussion

## Overview

During the kickoff discussions, David raised the topic of alternate domains he owns and their potential SEO implications for the redesign project.

---

## Key Discussion Points

**Domain Migration (incorporate123.co → incorporate123.com):**
David asked whether replacing `incorporate123.co` with `incorporate123.com` would be a step backwards from an SEO perspective, given that `.co` has the existing history.

**BIT Studios' Assessment:**
- Pointing multiple domains to a single site **no longer provides SEO benefit**. It only helps if people already know those domains or if valuable content is being created on them separately.
- Migrating from `.co` to `.com` would likely cause a **temporary dip** but should recover within **1–3 months**.
- With **301 redirects** on all URLs, the transition should be smooth — especially since URLs are already being restructured as part of the redesign, making it essentially "a wash."
- The team will confirm specifics with Irish (BIT Studios' SEO lead).

**Resolution:**
All domains can be pointed to the primary domain. The team will review the full list for any residual SEO value.
